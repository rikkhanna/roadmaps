Think of a 3D array as a book. The book has pages (axis 0), each page has rows (axis 1), each row has columns (axis 2). Shape (4, 3, 5) = 4 pages, 3 rows per page, 5 columns per row.

Real example — a colour image is a 3D array
Every image your ML model sees is stored as a 3D array: (height, width, channels). The 3 channels are Red, Green, Blue.
A 4×4 pixel image has shape (4, 4, 3).
axis 0 = row of pixels (height = 4)
axis 1 = column of pixels (width = 4)
axis 2 = colour channel (3: R, G, B)

image[2, 1, 0] = the Red value of the pixel at row 2, column 1. 
image = np.zeros((4, 4, 3), dtype=np.uint8)   # black image
image[0, :, 0] = 255  # top row → red channel → full red
image[:, :, 2] = 128  # all pixels → blue channel → half blue
image[2, 1, 0]        # Red value at row 2, col 1
Add one more dimension — a batch of images
ML models don't process one image at a time. They process a batch — many images stacked together. That makes it 4D.
1D (5,) — a list of 5 numbers
2D (3, 5) — a table: 3 rows, 5 columns
3D (H, W, 3) — one colour image
4D (32, H, W, 3) — a batch of 32 colour images

Each extra dimension is just another level of "stacking". A 3D array is a stack of 2D grids. A 4D array is a stack of 3D arrays. 
# What your model actually receives per training step:
batch = np.zeros((32, 224, 224, 3))  # 32 images, 224×224, RGB

batch.shape      # (32, 224, 224, 3)
batch[0].shape   # (224, 224, 3)  ← first image
batch[0, :, :, 0].shape  # (224, 224) ← red channel of first image
How to read and write values — indexing a 3D array
You provide one index per dimension, separated by commas. Think: [page, row, column].
arr[1] → give me page 1 (the whole grid) → shape (3, 5)
arr[1, 2] → page 1, row 2 (one row) → shape (5,)
arr[1, 2, 3] → page 1, row 2, column 3 → one number
arr[:, 0, :] → all pages, only row 0, all columns → shape (4, 5)
arr[:, :, 2] → all pages, all rows, only column 2 → shape (4, 3) 
arr = np.arange(60).reshape(4, 3, 5)

arr[1, 2, 3]   # → 38  (one number)
arr[0]         # → 2D grid, shape (3,5)
arr[:, 0, :]   # → all first rows, shape (4,5)
arr.shape      # (4, 3, 5)
arr.ndim       # 3
