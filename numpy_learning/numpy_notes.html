NumPy Cheat Sheet

Essential Array Operations & Manipulations

np.insert

Inserts values along a given axis before the specified indices.

Parameters
  • arr array_like: The input array where you want to insert the values.
  • obj int, slice or sequence of ints: The index (or indices) before which the values will be inserted.
    • Single int: Inserts before this specific index.
    • Sequence/list of ints: Inserts before multiple indices.
    • Slice: Inserts before the indices defined by a slice object.
  • values array_like: The values to insert. If it's an array, it must be broadcastable to the shape of the insertion. Will be cast to the type of arr if different.
  • axis int, optional: The axis along which to insert the values.
    • None (default): The input array is flattened first, and a 1D array is returned.
    • 0: Inserts rows (for 2D arrays).
    • 1: Inserts columns (for 2D arrays).
Return Value
  • out ndarray: A copy of arr with values inserted.

np.append & np.delete

Basic array element removal and addition operations.

  • np.append(arr, values): Appends value at the end of the array.
  • np.delete(arr, index): Deletes the element at a specified index.

Understanding Axis & Sorting

How dimensions are handled in 2D arrays.

  • axis=0: Means columns (vertically).
  • axis=1: Means rows (horizontally).
# 2D array sorting example
ints_2dim = np.array([[3, 2, 5, 7], 
                      [5, 0, 8, 3]])

# Sorts elements of each row (default axis=-1)
print(np.sort(ints_2dim))

np.arange

Creates an array with evenly spaced values within a given interval. Similar to Python's built-in range() function, but returns an ndarray.

numpy.arange([start, ]stop, [step, ]dtype=None)
Parameters
  • start number, optional: The starting value of the interval. Includes this value. Default: 0
  • stop number: The end of the interval. Half-open [start, stop), does not include this value.
  • step number, optional: Spacing between values. Default: 1. Can be negative to count backward or a float.
  • dtype data-type, optional: Output array type. Inferred by default.
Return Value
  • out ndarray: A 1D array of evenly spaced values.

np.linspace

Creates an array of evenly spaced numbers over a specified interval. Highly recommended over np.arange when working with floating-point numbers.

numpy.linspace(start, stop, num=50, endpoint=True, retstep=False, dtype=None)
Parameters
  • start array_like: The starting value of the sequence.
  • stop array_like: The end value of the sequence. Included by default.
  • num int, optional: The total number of elements to generate. Default: 50
  • endpoint bool, optional: Determines if stop is included.
    • True (default): stop is the last sample.
    • False: Interval is [start, stop). stop is excluded.
  • retstep bool, optional: If True, returns a tuple: (array, step_size). Default: False
  • dtype data-type, optional: Output array type. Defaults to float.
Return Value
  • out ndarray: An array of num equally spaced samples.
  • step float, optional: Size of spacing between samples (only if retstep=True).

np.full

Returns a new array of a given shape and type, filled with a specific fill_value.

numpy.full(shape, fill_value, dtype=None, order='C')
Parameters
  • shape int or sequence of ints: The shape of the new array. E.g., 5 or (2, 3).
  • fill_value scalar or array_like: The value to fill the array with.
  • dtype data-type, optional: Desired data-type. Infers from fill_value if None.
  • order {'C', 'F'}, optional: Memory layout. 'C' (Row-major) is default.
Return Value
  • out ndarray: An array with the specified shape, filled with the fill_value.

np.reshape

Gives a new shape to an array without changing its data. Total elements must remain constant.

numpy.reshape(a, newshape, order='C')
# Or as a method: arr.reshape(newshape)
Parameters
  • a array_like: The array you want to reshape.
  • newshape int or tuple of ints: The desired new shape.
    • Special value -1: Specify -1 for one dimension to let NumPy automatically calculate it.
  • order {'C', 'F', 'A'}, optional: Determines read/write order.
    • 'C' (default): C-like, row-major order (across rows).
    • 'F': Fortran-like, column-major order (down columns).
    • 'A': 'F' if Fortran contiguous in memory, else 'C'.
Return Value
  • reshaped_array ndarray: The reshaped array. Usually a view of the original array rather than a copy.
